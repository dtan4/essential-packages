# Essential packages

My essential packages

## Dependencies
* [Homebrew](http://brew.sh/) (Mac)
    * `ruby -e "$(curl -fsSL https://raw.github.com/Homebrew/homebrew/go/install)"`
* [Bundler](http://bundler.io/) (Ruby)
    * `gem install bundler`

## Usage
```sh
rake
brew bundle
bundle install
```
